# Groupe Nova Phénix International - Site Web Officiel

Site web vitrine pour le Groupe Nova Phénix International, un groupe multisectoriel opérant en Algérie, au Sénégal et en Afrique de l'Ouest.

## 🚀 Démarrage Rapide

1. Ouvrez `index.html` dans votre navigateur web
2. Pour le déploiement, consultez `DEPLOY.md` ou `DEPLOY_O2SWITCH.md`

## 📁 Structure du Projet

```
Nova Phénix Site/
├── index.html              # Page principale
├── dist/                   # Version de production
├── Dercavo Réalisations/   # Photos galerie Dercavo
├── Image Event/            # Photos événements
├── Image Home/             # Photos aménagement intérieur
├── Image Travel/           # Photos voyage
├── Logo Nova Phenix *.png  # Logos des filiales
└── *.webp, *.jpg           # Logos partenaires
```

## 🏢 Les 5 Filiales

- **Nova Phénix Events** - Événementiel & Salons professionnels
- **Nova Phénix Consulting** - Conseil & Formation
- **Nova Phénix Trade** - Import-Export & Commerce international
- **Nova Phénix Home** - Architecture d'intérieur & Rénovation
- **Nova Phénix Travel** - Voyage & Immigration

## 🛠 Technologies Utilisées

- **HTML5** - Structure sémantique
- **Tailwind CSS** (CDN) - Stylisation responsive
- **Font Awesome 6.4** - Icônes
- **Google Fonts** - Montserrat & Open Sans
- **AOS** - Animations au scroll
- **Formspree** - Gestion du formulaire de contact

## 🔧 Maintenance

### Supprimer les fichiers parasites macOS
```bash

find . -name ".DS_Store" -delete
```

Utiliser un outil comme ImageOptim ou Squoosh pour compresser les images volumineuses.
